Hierologia

An open academic API for the comparative study of sacred traditions. Every entry is sourced, neutral, and versioned.

    Select an entry to begin.

    Hierologia preserves theological, historical, and sociological knowledge across religious traditions. The first tradition is Islamic theology (kalam). Each concept, figure, and school links to the others, forming a graph of belief - and every claim carries its sources.

    This demo is served entirely from static JSON. Any entry below is also a live API endpoint.
